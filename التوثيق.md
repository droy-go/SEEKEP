## مقدمة

SEEKEP (Smart Expressive Efficient Knowledge Execution Platform) هي لغة برمجة عربية عالية المستوى. تم تصميمها لتكون:

- **سهلة التعلم**: بناء جمل عربي بسيط
- **قوية**: دعم البرمجة كائنية التوجه والبرمجة الوظيفية
- **سريعة**: ترجمة إلى بايتكود وتنفيذ عبر جهاز افتراضي
- **مستقلة**: مكتوبة من الصفر بلغة C

### لماذا SEEKEP؟

- برمجة باللغة العربية
- لا تعتمد على مكتبات خارجية
- مفتوحة المصدر بالكامل
- مناسبة للتعليم والتطبيقات العملية

## البدء السريع

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/droy-go/seekep.git
cd seekep

# البناء
make

# التثبيت (اختياري)
sudo make install
```

### أول برنامج

```seekep
# حفظ في ملف: مرحبا.سكيب
اطبع("مرحباً بالعالم!")
```

التشغيل:
```bash
./bin/seekep مرحبا.سكيب
```

```bash
$ ./bin/seekep -i
```

## تدفق التحكم

### عبارة إذا

```seekep
متغير الدرجة = 85

إذا (الدرجة >= 90) {
    اطبع("ممتاز")
} وإلا إذا (الدرجة >= 80) {
    اطبع("جيد جداً")
} وإلا إذا (الدرجة >= 70) {
    اطبع("جيد")
} وإلا إذا (الدرجة >= 60) {
    اطبع("مقبول")
} وإلا {
    اطبع("راسب")
}
```

### العملية الثلاثية

```seekep
متغير العمر = 20
متغير الفئة = (العمر >= 18) ? "بالغ" : "قاصر"
```

### حلقة أثناء

```seekep
متغير عداد = 1
أثناء (عداد <= 10) {
    اطبع("العدد: " + نص(عداد))
    عداد = عداد + 1
}
```

### حلقة لكل (المدى)

```seekep
# من 1 إلى 10
لكل (i في المدى(1، 11)) {
    اطبع("التكرار: " + نص(i))
}

# مع خطوة
لكل (i في المدى(0، 11، 2)) {
    اطبع(i)  # 0, 2, 4, 6, 8, 10
}

# تنازلي
لكل (i في المدى(10، 0، -1)) {
    اطبع(i)  # 10, 9, 8, ..., 1
}
```

### حلقة لكل (القائمة)

```seekep
متغير فواكه = ["تفاح"، "موز"، "برتقال"]

لكل (فاكهة في فواكه) {
    اطبع("أحب " + فاكهة)
}
```

### حلقة لكل (القاموس)

```seekep
متغير شخص = {"الاسم": "أحمد"، "العمر": 30}

لكل (مفتاح في المفاتيح(شخص)) {
    اطبع(مفتاح + ": " + نص(شخص[مفتاح]))
}
```

### توقف واستمر

```seekep
# توقف - إيقاف الحلقة
لكل (i في المدى(1، 11)) {
    إذا (i == 5) {
        توقف
    }
    اطبع(i)  # 1, 2, 3, 4
}

# استمر - تخطي التكرار الحالي
لكل (i في المدى(1، 11)) {
    إذا (i % 2 == 0) {
        استمر
    }
    اطبع(i)  # 1, 3, 5, 7, 9
}
```

## الدوال

### تعريف دالة

```seekep
dالة تحية(الاسم) {
    أرجع "مرحباً، " + الاسم + "!"
}

اطبع(تحية("محمد"))  # "مرحباً، محمد!"
```

### دالة مع معاملات متعددة

```seekep
dالة جمع(أ، ب، ج = 0) {
    أرجع أ + ب + ج
}

اطبع(جمع(1، 2))      # 3
اطبع(جمع(1، 2، 3))   # 6
```

### دالة بدون إرجاع

```seekep
dالة اطبع_معلومات(الاسم، العمر) {
    اطبع("الاسم: " + الاسم)
    اطبع("العمر: " + نص(العمر))
}

اطبع_معلومات("أحمد"، 25)
```

### دالة متكررة

```seekep
dالة عامل(ن) {
    إذا (ن <= 1) {
        أرجع 1
    }
    أرجع ن * عامل(ن - 1)
}

اطبع(عامل(5))  # 120
```

### دالة مجهولة (Lambda)

```seekep
متغير مضاعف = دالة(س) => س * 2
متغير جمع = دالة(أ، ب) => أ + ب

اطبع(مضاعف(5))     # 10
اطبع(جمع(3، 4))    # 7
```

### دالة عالية المستوى

```seekep
dالة تطبيق(قائمة، دالة) {
    متغير النتيجة = []
    لكل (عنصر في قائمة) {
        أضف(النتيجة، دالة(عنصر))
    }
    أرجع النتيجة
}

متغير أرقام = [1، 2، 3، 4، 5]
متغير مضاعفات = تطبيق(أرقام، دالة(س) => س * 2)
اطبع(مضاعفات)  # [2, 4, 6, 8, 10]
```

### Closure

```seekep
dالة منشئ_عداد() {
    متغير العداد = 0
    أرجع دالة() {
        العداد = العداد + 1
        أرجع العداد
    }
}

متغير عداد1 = منشئ_عداد()
متغير عداد2 = منشئ_عداد()

اطبع(عداد1())  # 1
اطبع(عداد1())  # 2
اطبع(عداد2())  # 1
اطبع(عداد1())  # 3
```

## القوائم والقواميس

### القوائم

```seekep
# إنشاء قائمة
متغير أرقام = [1، 2، 3، 4، 5]
متغير نصوص = ["أحمد"، "محمد"، "علي"]
متغير مختلط = [1، "نص"، صحيح، [1، 2]]

# الوصول للعناصر
اطبع(أرقام[0])      # 1 (أول عنصر)
اطبع(أرقام[-1])     # 5 (آخر عنصر)

# التعديل
أرقام[0] = 10

# الإضافة
أضف(أرقام، 6)
أدخل(أرقام، 0، 0)   # إدراج في البداية

# الإزالة
اسحب(أرقام)         # إزالة الأخير
اسحب(أرقام، 0)      # إزالة من فهرس

# البحث
متغير فهرس = أرقام.فهرس(3)  # -1 إذا لم يوجد

# الفرز والعكس
رتب(أرقام)
اعكس(أرقام)

# النسخ
متغير نسخة = انسخ(أرقام)

# معلومات
اطبع(الطول(أرقام))
```

### القواميس

```seekep
# إنشاء قاموس
متغير شخص = {
    "الاسم": "أحمد",
    "العمر": 30,
    "المدينة": "القاهرة"
}

# الوصول
اطبع(شخص["الاسم"])

# التعديل
شخص["العمر"] = 31

# الإضافة
شخص["المهنة"] = "مبرمج"

# الإزالة
احذف(شخص، "المدينة")

# التحقق
إذا ("الاسم" في شخص) {
    اطبع("الاسم موجود")
}

# الحصول على المفاتيح والقيم
متغير مفاتيح = المفاتيح(شخص)
متغير قيم = القيم(شخص)
متغير أزواج = الأزواج(شخص)

# التكرار
لكل (مفتاح في المفاتيح(شخص)) {
    اطبع(مفتاح + ": " + نص(شخص[مفتاح]))
}
```

## الأصناف والكائنات

### تعريف صنف

```seekep
صنف شخص {
    دالة init(الاسم، العمر) {
        هذا.الاسم = الاسم
        هذا.العمر = العمر
    }
    
    دالة تقديم() {
        اطبع("أنا " + هذا.الاسم + "، عمري " + نص(هذا.العمر))
    }
    
    دالة عيد_ميلاد() {
        هذا.العمر = هذا.العمر + 1
        أرجع هذا.العمر
    }
}
```

### إنشاء كائنات

```seekep
متغير أحمد = جديد شخص("أحمد"، 25)
متغير محمد = جديد شخص("محمد"، 30)

أحمد.تقديم()
محمد.تقديم()

أحمد.عيد_ميلاد()
اطبع(أحمد.العمر)  # 26
```

### الوراثة

```seekep
صنف طالب: شخص {
    دالة init(الاسم، العمر، التخصص) {
        هذا.الاسم = الاسم
        هذا.العمر = العمر
        هذا.التخصص = التخصص
        هذا.الدرجات = []
    }
    
    دالة إضافة_درجة(درجة) {
        أضف(هذا.الدرجات، درجة)
    }
    
    دالة المتوسط() {
        إذا (الطول(هذا.الدرجات) == 0) {
            أرجع 0
        }
        متغير المجموع = 0
        لكل (درجة في هذا.الدرجات) {
            المجموع = المجموع + درجة
        }
        أرجع المجموع / الطول(هذا.الدرجات)
    }
    
    دالة تقديم() {
        اطبع("أنا " + هذا.الاسم + "، طالب " + هذا.التخصص)
    }
}

متغير طالب1 = جديد طالب("علي"، 20، "علوم الحاسب")
طالب1.إضافة_درجة(85)
طالب1.إضافة_درجة(90)
اطبع(طالب1.المتوسط())
```

### الخصائص الخاصة

```seekep
صنف حساب_بنكي {
    دالة init(الصاحب، الرصيد_الأولي = 0) {
        هذا.الصاحب = الصاحب
        هذا.الرصيد_الخاص = الرصيد_الأولي
    }
    
    دالة إيداع(المبلغ) {
        إذا (المبلغ > 0) {
            هذا.الرصيد_الخاص = هذا.الرصيد_الخاص + المبلغ
            أرجع صحيح
        }
        أرجع خطأ
    }
    
    دالة سحب(المبلغ) {
        إذا (المبلغ > 0 و المبلغ <= هذا.الرصيد_الخاص) {
            هذا.الرصيد_الخاص = هذا.الرصيد_الخاص - المبلغ
            أرجع صحيح
        }
        أرجع خطأ
    }
    
    دالة الرصيد() {
        أرجع هذا.الرصيد_الخاص
    }
}

متغير حساب = جديد حساب_بنكي("أحمد"، 1000)
حساب.إيداع(500)
حساب.سحب(200)
اطبع(حساب.الرصيد())  # 1300
```

## المكتبة القياسية

### الرياضيات

| الدالة | الوصف | مثال |
|--------|-------|------|
| `جذر(س)` | الجذر التربيعي | `جذر(16)` → 4 |
| `أس(س، ص)` | الأس | `أس(2، 3)` → 8 |
| `جيب(س)` | جيب الزاوية | `جيب(0)` → 0 |
| `جيب_تام(س)` | جيب التمام | `جيب_تام(0)` → 1 |
| `ظل(س)` | الظل | `ظل(0)` → 0 |
| `لوغاريتم(س)` | اللوغاريتم الطبيعي | `لوغاريتم(1)` → 0 |
| `لوغاريتم10(س)` | اللوغاريتم العشري | `لوغاريتم10(100)` → 2 |
| `أس_الطبيعي(س)` | e^x | `أس_الطبيعي(0)` → 1 |
| `أرض(س)` | التقريب للأسفل | `أرض(3.7)` → 3 |
| `سقف(س)` | التقريب للأعلى | `سقف(3.2)` → 4 |
| `تقريب(س)` | التقريب | `تقريب(3.5)` → 4 |
| `قيمة_مطلقة(س)` | القيمة المطلقة | `قيمة_مطلقة(-5)` → 5 |
| `أصغر(...)` | الأصغر | `أصغر(3، 1، 4)` → 1 |
| `أكبر(...)` | الأكبر | `أكبر(3، 1، 4)` → 4 |
| `عشوائي()` | عشوائي [0,1) | `عشوائي()` |
| `عشوائي(من، إلى)` | عشوائي صحيح | `عشوائي(1، 10)` |

### النصوص

| الدالة | الوصف | مثال |
|--------|-------|------|
| `نص(قيمة)` | تحويل إلى نص | `نص(42)` → "42" |
| `كبير(نص)` | أحرف كبيرة | `كبير("مرحبا")` → "مرحبا" |
| `صغير(نص)` | أحرف صغيرة | `صغير("مرحبا")` → "مرحبا" |
| `تقليم(نص)` | إزالة المسافات | `تقليم("  نص  ")` → "نص" |
| `قسم(نص، فاصل)` | تقسيم | `قسم("أ,ب,ت"، ",")` |
| `اربط(قائمة، فاصل)` | ربط | `اربط(["أ"، "ب"]، "-")` |
| `استبدل(نص، قديم، جديد)` | استبدال | `استبدل("مرحبا"، "حب"، "ساف")` |
| `يبدأ_بـ(نص، بادئة)` | التحقق | `يبدأ_بـ("مرحبا"، "مر")` |
| `ينتهي_بـ(نص، لاحقة)` | التحقق | `ينتهي_بـ("مرحبا"، "با")` |
| `ابحث(نص، جزء)` | البحث | `ابحث("مرحبا"، "حب")` → 2 |
| `حرف(رمز)` | من رمز ASCII | `حرف(65)` → "A" |
| `ترميز(حرف)` | إلى رمز ASCII | `ترميز("A")` → 65 |

### القوائم

| الدالة | الوصف |
|--------|-------|
| `أضف(قائمة، عنصر)` | إضافة في النهاية |
| `أدخل(قائمة، فهرس، عنصر)` | إدراج في فهرس |
| `اسحب(قائمة، فهرس = -1)` | إزالة وإرجاع |
| `احذف(قائمة، فهرس)` | حذف عنصر |
| `امسح(قائمة)` | مسح الكل |
| `رتب(قائمة)` | ترتيب |
| `اعكس(قائمة)` | عكس |
| `انسخ(قائمة)` | نسخ |
| `الطول(قائمة)` | عدد العناصر |

### القواميس

| الدالة | الوصف |
|--------|-------|
| `احذف(قاموس، مفتاح)` | حذف مفتاح |
| `امسح(قاموس)` | مسح الكل |
| `المفاتيح(قاموس)` | قائمة المفاتيح |
| `القيم(قاموس)` | قائمة القيم |
| `الأزواج(قاموس)` | قائمة الأزواج |
| `الطول(قاموس)` | عدد الإدخالات |

### الملفات

| الدالة | الوصف | مثال |
|--------|-------|------|
| `افتح(مسار، نمط)` | فتح ملف | `افتح("test.txt"، "w")` |
| `اقرأ(مسار)` | قراءة ملف | `اقرأ("test.txt")` |
| `اكتب(مسار، محتوى)` | كتابة ملف | `اكتب("test.txt"، "نص")` |
| `أغلق(ملف)` | إغلاق ملف | `أغلق(ملف)` |
| `موجود(مسار)` | التحقق | `موجود("test.txt")` |
| `احذف_ملف(مسار)` | حذف ملف | `احذف_ملف("test.txt")` |
| `أعد_تسمية(قديم، جديد)` | إعادة تسمية | `أعد_تسمية("old.txt"، "new.txt")` |
| `أنشئ_مجلد(مسار)` | إنشاء مجلد | `أنشئ_مجلد("test")` |
| `احذف_مجلد(مسار)` | حذف مجلد | `احذف_مجلد("test")` |
| `المحتويات(مسار)` | قائمة المحتويات | `المحتويات(".")` |

## الأخطاء والاستثناءات

### أنواع الأخطاء

1. **أخطاء ترجمة**: أخطاء في بناء الجمل
2. **أخطاء زمنية**: أخطاء أثناء التنفيذ

### معالجة الأخطاء

```seekep
# محاكاة try-catch باستخدام إذا
dالة قسمة_آمنة(أ، ب) {
    إذا (ب == 0) {
        اطبع("خطأ: قسمة على صفر")
        أرجع فارغ
    }
    أرجع أ / ب
}

متغير نتيجة = قسمة_آمنة(10، 0)
إذا (نتيجة != فارغ) {
    اطبع("النتيجة: " + نص(نتيجة))
}
```
